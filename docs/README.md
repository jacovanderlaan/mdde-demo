## Quick Reference

### Repository Purpose
The mdde-demo repository is an **educational subset** of MDDE that:
- Proves MDDE concepts are real and runnable
- Provides working code examples for articles
- Invites experimentation without exposing enterprise features

### Target Audience
- Readers of Medium articles who want hands-on experience
- Data engineers evaluating MDDE concepts
- Workshop participants needing a starting point

### What It Is NOT
- A full implementation of MDDE
- Production-ready code
- A replacement for the full framework

---

## Key Principles

1. **Clarity over Completeness** - Simple, readable code that demonstrates concepts
2. **Educational Focus** - Every feature should teach something
3. **Article Alignment** - Demo code should reference specific articles
4. **Minimal Dependencies** - Only sqlglot, duckdb, pyyaml
